# AGENTS.md

이 파일은 Claude Code 외 다른 에이전트(Codex CLI, Gemini CLI, Cursor 등)가 이 프로젝트를 다룰 때 공통으로 따라야 할 규칙이다. CLAUDE.md와 중복되는 내용은 CLAUDE.md를 우선한다.

## 작업 범위 제한

이 프로젝트에서 에이전트가 **하지 말아야 할 것**:

- `docs/references.md`의 인용 출처를 임의로 변경하지 말 것 (사실 검증 필요)
- 슬라이드 50장 한도를 넘기지 말 것 (시간 압박)
- 다삼솔루션, ARS, Jetson, KSL 등 사용자의 다른 프로젝트와 관련된 내용 추가 금지
- 다크/라이트/미드나이트 외의 새 테마 추가 금지 (스코프 크리프 방지)

## 서브에이전트별 역할

이 프로젝트에서 서브에이전트를 spawn할 때 권장 분담:

```
slide-writer    → 슬라이드 콘텐츠 작성 (한국어/영어 키워드)
visualizer      → SVG/Mermaid 다이어그램 제작
fact-checker    → 통계/인용의 출처 확인
designer        → CSS/스타일 다듬기
```

각 서브에이전트는 자기 worktree에서 작업한 후 main으로 머지.

## 검증 체크리스트

PR 머지 전 확인:

- [ ] 슬라이드 텍스트가 한 화면 절반 이상을 채우지 않는가
- [ ] 인용에 출처가 있는가 (references.md 참조)
- [ ] 한국어와 영어 키워드 균형이 맞는가
- [ ] 16:9에서 깨지지 않는가
- [ ] 3가지 테마 모두에서 가독성 OK인가
